# RFDD-LaTeX-Report

同济大学交通学院 · 智能轨道交通 · 实验报告 LaTeX 源码

---

## 项目简介

本仓库为智能轨道交通第二次实验报告（大作业）的 LaTeX 源代码，基于 RFDD（Railway Fastener Defect Dataset）数据集，对比研究 YOLOv8s / YOLOv8m、Faster R-CNN、RetinaNet 等目标检测算法在铁路扣件缺陷检测中的性能。
