\section{实验背景与目的}
\par{}
铁路扣件是连接钢轨与轨枕的重要部件。扣件缺失、断裂、反装、位移、变形等问题会影响线路安全。本作业使用 RFDD（Railway Fastener Defect Dataset）数据集，训练目标检测模型识别铁路扣件状态，并比较 YOLO 与其他目标检测算法在精度、速度和工程适用性上的差异。
\par{}
本次实验的目的如下：
\begin{enumerate}[\bf(1)]
    \vspace{-2.4mm}
    \item 面向现实工程问题，要求学生完成从数据读取、标注解析、可视化验证到模型训练、测试评估、对比分析的一系列任务，掌握目标检测的全流程操作；
          \vspace{-3.5mm}
    \item 促进学生理解YOLO模型的基本算法原理，并能对超参数进行合理调整、对模型性能进行多指标评价；
          \vspace{-3.5mm}
    \item 培养学生使用不同模型解决同一个工程问题，并分析各模型从原理到性能的差异的能力；
          \vspace{-3.5mm}
    \item 培养学生在实际工程情境下，理解和应对样本缺陷的能力。
          \vspace{-3.5mm}
\end{enumerate}

本目录中的 RFDD 数据集包含铁路扣件图像和目标检测标注：

    \vspace{-2.4mm}
    \item \textbf{训练与验证集：train\&val/}
          \vspace{-1.5mm}
          \par{}
          \hspace{1.65em}
          图像文件：train\&val/images/RFDD\_Train\&val\_Images\_dedup.h5
          \vspace{-1.5mm}
          \par{}
          \hspace{1.65em}
          标注文件：train\&val/labels/RFDD\_Train\&val\_Labels.h5
          \vspace{-1.5mm}
          \vspace{-2mm}
    \item \textbf{测试集：test/}
          \vspace{-1.5mm}
          \par{}
          \hspace{1.65em}

          \vspace{-1.5mm}

          标注目录：test/labels/
